# Furhat Skill: Matcha Mochi Tasting Promotion 🤖🍵

This is a conversational Furhat robot skill designed to promote mochi tasting and assist customers in purchasing different flavors, with a special discount campaign for matcha flavor.

## ✨ Features

- Friendly interaction with customers using speech
- Supports natural language understanding (NLU) for flavor and quantity
- Guides users through a "buy 2 get 1 free" promotion for matcha mochi
- Handles hesitation, rejection, and unexpected responses

## 🧠 Technologies Used

- Kotlin with Furhat SDK
- Custom NLU intents and entities
- State machine-driven dialogue flow
- Works with virtual or physical Furhat robot

## 🚀 How to Run

1. Open the project in IntelliJ IDEA
2. Make sure the Furhat Launcher and Virtual Robot are running
3. Run the project using the green "play" button
4. Interact with the robot and monitor logs, speech, and attention in the Furhat GUI
